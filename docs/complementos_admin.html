Gestión de Complementos

Los complementos son adicionales opcionales que la agencia ofrece por viaje, fuera del precio base.

Ejemplos: Seguro de cancelación (S/35), Menú celíaco (S/15), Excursión al lago (S/40), Kit de viaje (S/25).

Tipos de complemento

Tipo Qué incluye Etiqueta
seguro Seguros de viaje, cancelación, accidentes, equipaje SEGURO
menu Menús especiales: celíaco, vegetariano, alergias MENÚ
actividad_opcional Excursión extra, taller, visita no incluida en el precio base ACTIVIDAD
extra Camiseta del viaje, fotos, recuerdo, kit de viaje EXTRA

Modelos

1. Complemento — Catálogo maestro

Complemento
├── nombre       — ej. "Seguro Cancelación", "Menú Celíaco"
├── tipo         — seguro | menu | actividad_opcional | extra
├── descripcion  — texto explicativo para el padre
└── activo       — si está disponible globalmente

Se administra en: Admin > Viajes > Complementos

2. ComplementoViaje — Puente con precio

ComplementoViaje
├── viaje         — FK → Viaje
├── complemento   — FK → Complemento
├── precio        — ej. S/35.00
└── activo        — si está disponible para este viaje

Se administra como inline dentro del Viaje (sección "Complementos por viaje") o directamente en Admin > Viajes > Complementos por viaje.

3. ComplementoContratado — Lo que contrató cada familia

ComplementoContratado
├── inscripcion        — FK → Inscripcion
├── complemento_viaje  — FK → ComplementoViaje
└── cantidad           — ej. 2

Visible como inline de solo lectura dentro de cada Inscripción.

Flujo completo

  1. Crear el catálogo — Admin > Viajes > Complementos: crear "Seguro Básico", "Menú Vegano", etc.
  2. Asignar a un viaje — Editar el Viaje, sección "Complementos por viaje": elegir complemento y fijar precio
  3. El padre contrata — Durante la inscripción (frontend), los complementos activos aparecen como opcionales
  4. Consultar en admin — En la Inscripción, inline "Complementos contratados" muestra lo elegido por cada familia
  5. Exportar — Los complementos contratados se incluyen en los reportes para proveedores (hotel, restaurante)
